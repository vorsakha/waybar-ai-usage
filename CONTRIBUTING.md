# Contributing

Contributions are welcome through pull requests.

1. Fork the repository and create a focused branch.
2. Keep changes small and avoid committing credentials, cache files, or local configuration.
3. Run the test suite:

   ```bash
   python3 -m unittest discover -s tests -v
   ```

4. Open a pull request describing what changed and how it was verified.

`main` is protected. Changes must pass CI and be reviewed before they are merged.
